! Для работы необходимо установить библиотеку pygame !

-------------------------------------------------------------------------------------------------------------------------------------------------------------
Описание: Два игрока поочередно делают ходы на поле 3х3, пытаясь составить линию из трех своих символов (крестиков или ноликов).
Особенности реализации: Массив для хранения состояния игрового поля, функции для вывода поля, проверки победы и обработки хода.
Основные элементы: Циклы для итерации ходов и проверка условий
-------------------------------------------------------------------------------------------------------------------------------------------------------------
Инструкция для запуска:

PyCharm:
Перенесите файл "gameMarat.py" в деректорию проекта, предварительно установив Python на компьютер, или создайте новый проект в PyCharm.
В левой строчке снизу выберете ярлык Python Packages и нажмите на него, в строке поиска введите pygame и скачайте библиотеку выбрав одну из последних версий.
По завершению загрузки, откройте файл "gameMarat.py" в меню проекта, выберите его в верхней панели правее середины (или там же выберите строчку Current File)
Нажмите на треугольник правее и у вас запустится игра.

IDLE:
Попробуйте открыть консоль и использовать эту команду: pip install pygame
Если не получилось, попробуйте для начала установить библиотеку pip с официального сайта или по этой ссылке: https://bootstrap.pypa.io/get-pip.py
После установки нажмите на файл "gameMarat.py" правой кнопкой мыши и откройте его нажав Edit with IDLE.
После, нажмите F5 или на кнопку Run и у вас запустится игра.
